# Munkres-Topology-Notes

## 阅读建议 (Reading Guide)

本笔记主要使用 **Obsidian** 编写。由于 GitHub 自带的 Markdown 渲染引擎与 Obsidian 存在语法差异，部分复杂的 LaTeX 数学公式在网页端可能会显示 `Unable to render expression`。

> **提示：** 为了获得最佳阅读体验，建议下载后使用 Obsidian 打开。
